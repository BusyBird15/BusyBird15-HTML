Welcome to Simplecode!

The simplest, easiest, lightest code editor on the web.


At the top you will find the toolbar, which consists of:
* Theme editor: for changing the code editor themes.
* Font editor: for changing the code editor font.
* Highlight language: for setting the syntax highlighting language.
* Download button: to download your code file.
* Upload button: to upload a file for editing.
* Settings button: to set other settings like font size.
* Help button: to re-open this dialog again.


Happy coding!
