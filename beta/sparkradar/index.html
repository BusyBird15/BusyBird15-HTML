Thanks for checking out my radar. Spark Radar was a project I started in 2024 because I love tracking the weather and programming. I made the entire thing open source because finding the radar data was really hard to do. Github!

How does it work?

The principle of Spark Radar is actually very simple. I found a tile server used by the NWS's radar. After passing it a bounding box and station, it returns an image which simply gets placed on the map!
